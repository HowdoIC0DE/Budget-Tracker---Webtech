# Budget Tracker

Budget Tracker ist eine Webanwendung zur Verwaltung von Einnahmen und Ausgaben.

## Projektbeschreibung
In diesem Projekt wird eine Anwendung entwickelt, mit der Nutzer ihre finanziellen Transaktionen verwalten können.

Für Milestone 1 wurde eine erste Spring Boot Anwendung erstellt, die über eine REST-API Beispiel-Transaktionen zurückgibt.

## Tech Stack
- Spring Boot (Backend)
- Vue (Frontend)
- PostgreSQL (Datenbank geplant)
